# **App Name**: IdeaForge Blueprint

## Core Features:

- Idea Submission Interface: Provide a clean, focused input area for users to submit their 1-2 line business ideas.
- Blueprint Generation Engine: Utilize LLM orchestration as a tool to chain multiple calls, synthesizing input into a comprehensive multi-module startup blueprint.
- Interactive Report Viewer: Display the generated blueprint modules in a structured, navigable, and highly readable format, supporting markdown content.
- Temporary Session Caching: Implement a robust in-memory or JSON-based temporary caching strategy to handle user inputs and generated reports for the current session.
- Blueprint Export Option: Enable users to download or export their generated startup blueprints in common formats like Markdown or PDF.
- Prompt Refinement Assistant: An AI tool that provides suggestions and feedback to users to help optimize their initial idea prompt for clearer and more effective blueprint generation.

## Style Guidelines:

- Primary color: Deep indigo (#24478F), evoking intelligence, trust, and professionalism.
- Background color: Very light blue-grey (#F3F4F6), for a clean, calm, and readable workspace, visibly of the same hue as the primary but heavily desaturated and bright.
- Accent color: Vibrant turquoise (#22A9C3), providing an energetic and clear contrast for interactive elements and highlights, analogous to the primary color.
- Headline and body font: 'Inter' (sans-serif), chosen for its modern, clean, and highly readable characteristics, perfect for displaying detailed reports and ensuring a 'Clean SaaS' aesthetic.
- Utilize a minimalist, line-based icon set to complement the 'Clean SaaS' design, ensuring clarity and unobtrusive visual guidance.
- Adopt a structured, multi-column layout for the report viewer, enhancing the organization and readability of comprehensive blueprint modules with clear sectioning.
- Implement subtle and purposeful animations, such as smooth transitions between report sections and discreet hover effects, to improve user experience without distraction.
